CSS 布局(flex, grid), 可以应用在 html, body 或者任意容器元素上, 外部容器的布局会影响内层的布局

viewport 设置, (<meta name="viewport">); 这个可以控制 html/body 的渲染宽度, 让布局与设备屏幕匹配

总结来说, window是最外层, 但不是HTML元素
页面布局主要作用在 <html>, <body> 以及容器元素上, 子元素的布局会受到父容器的影响
学 flex/grid, 可以从<body>或者顶层容器开始练习

层级结构概览

window 浏览器窗口

最外层是浏览器 window 对象, 这是一个 JavaScript 对象, 代表浏览器窗口, 可以通过 window.innerWidth / innerHeight 获取尺寸

这不是一个 HTML 元素, 但他是页面布局的最外层"容器"

document 整个 html 文档

document, 整个 html 文档, 代表页面 dom 树的根节点, root

可以通过 document.documentElement 访问 <html> 元素

html 文档的根元素

可以设置height:100%, display:flex 等布局属性

常用作 全局布局容器

head 文档头部

包含文档的元数据, 如 title, meta, link 等

常用作 全局样式和元数据容器

body 文档主体

页面可见内容的容器

可以设置背景, flex/grid布局, margin/padding等

页面内部的 div, section, header, main, footer 等元素

flex, grid, float 等布局大多作用在这些元素上

页面内部的 p, img, button 等元素

通过 css 可以控制它们在父容器里的排列和大小
